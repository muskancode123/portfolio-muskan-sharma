# Personal Portfolio Website

A responsive and modern portfolio website built using **Django REST Framework** and **React**.

## Features
- Responsive and user-friendly design
- Project showcase section
- Skills and technologies section
- Education and achievements
- Contact form with backend integration
- REST API powered by Django
- Dynamic frontend built with React

## Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Django
- Django REST Framework

### Database
- SQLite (Development)

## Purpose
This portfolio serves as a central platform to showcase my technical skills, academic projects, and software development experience while demonstrating full-stack development capabilities.
